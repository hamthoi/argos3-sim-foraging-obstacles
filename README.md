# argos3-sim-foraging-obstacles
This repository is not meant to be cloned and run. Instead, follow these simple steps:
1. Git clone ARGoS's example repository found on https://www.argos-sim.info/examples.php after installing ARGoS.
2. Put foraging_obstacles_loop_functions.cpp and foraging_obstacles_loop_functions.h in the loop_functions/foraging_loop_functions/ folder.
3. Put foraging_obstacles.argos in the experiments/ folder.
4. Replace loop_functions/foraging_loop_functions/CMakeLists.txt with the CMakeLists.txt in this repository.
5. Build and run, following their README.md.
